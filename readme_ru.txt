libtcod-duktape - это реализация функций библиотеки roguelike игр на ecmascript (тот же javascript) движке duktape. Идея движка - простая и быстрая разработка roguelike игр на ecmascript (проще говоря js), без портирования и пересборки бинарников. Логика игры пишется на js, тяжелые для duktape расчёты реализуются на си.
libtcod умеет выводить псевдографику, считать зону обзора, путь, генерировать карты высот, шум. - подробнее в документации.
duktape - движок ecmascript (E5/E5.1, некоторые фичи E6 и E7) написанный на си, компактный и легко встраиваемый.
На данный момент реализованны функции файловой системы: запись, чтение из файла, чтение списка каталогов, задержка и вывод в stdout.
Для libtcod реализована базовая работа с выводом на экран, цветами, шрифтами, работа с картой - расчёт зоны обзора, a*, линии.
На данный момент код пишется в основном на js, есть зачатки игрового фреймворка, наращивание функционала которого стимулируется посредством разработки roguelike шутера с управлением из unreal world и боёвкой из asciisector.
Собирается при помощи make под linux, под windows, скорее всего, надо портировать. При сборке линкуется на библиотеку libtcod.so, которая должна лежать в одном каталоге с бинарником.
